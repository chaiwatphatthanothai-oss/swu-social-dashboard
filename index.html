เชื่อมต่อ Google Sheets (Live Update)
กรอกข้อมูลใน Google Sheets แล้วกด Refresh — Dashboard อัปเดตทันที ไม่ต้องแก้ไขไฟล์ HTML
📋 วิธีตั้งค่า (ทำครั้งเดียว):
1. เปิด Google Sheets ที่ใช้กรอกข้อมูล → เมนู ส่วนขยาย → Apps Script
2. วางโค้ดจากไฟล์ GoogleAppsScript_Dashboard.js ที่ดาวน์โหลดมาจากโฟลเดอร์เดียวกับ dashboard.html
3. กด Deploy → New deployment → Web app → ตั้ง "Anyone" → คัดลอก URL
4. วาง URL ในช่องด้านบน แล้วกด "ทดสอบ"
🔄 ล้างข้อมูลและใช้ข้อมูลตัวอย่าง
